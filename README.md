# 크리스마스 카드

## Demo Link

🔗https://merrychristmas-2021.netlify.app/


편지를 작성하면 크리스마스 트리에 데코와 글이 추가됩니다.

- Firebase Realtime Database로 추가, 읽기, 삭제 구현
- Netlify로 배포
- 랜덤으로 데코 위치 생성
- JavaScript에서 TypeScript로 리팩토링

## Tech Stack

- TypeScipt
- React
- Redux
- Redux Toolkit
- Styled-Components
- Firebase Realtime Database
- Netlify
